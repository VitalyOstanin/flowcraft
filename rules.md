## 1. Общение с пользователем

Общаться с пользователем всегда на русском языке. Использовать обращение на вы с маленькой буквы без кавычек.

## 2. Создание документации

При создании md файлов:
- Добавлять TOC (содержание)
- Писать содержимое на русском языке
- Не использовать эмодзи

## 3. Работа с Git

Коммиты без разрешения делать НЕЛЬЗЯ!

## 4. Управление задачами

Выполненные задачи помечать в файле плана как выполненные.

## 5. Подход к решению задач

Все задачи доводить до конца. Не торопиться, выбирать правильное решение, а не быстрое.

## 6. Управление сложными задачами

Совсем сложные для немедленной реализации вещи записывать в файл TODO.md

## 7. Использование эмодзи

Не использовать эмодзи в проекте, за исключением зеленой галочки и красного креста в тестах и проверках.

## 8. Согласование с пользователем

При согласовании с пользователем разделить проблемы на отдельные пункты и проходить согласование отдельно по каждому пункту.
